API 使用说明

接口地址

POST /openapi/v1/generate

认证方式

在请求头中携带 API Key：

Authorization: Bearer ak_xxxxxxxxxxxxxxxx

请求参数

参数类型必填说明
promptstring是提示词
modestring否txt2img（文生图）或 img2img（图生图），默认 img2img
modelIdstring否模型：gpt-image / dashscope / jimeng 等
nnumber否生成数量（1-4，取决于模型）
hdboolean否超清模式
aspectRatiostring否比例，如 1:1、16:9、9:16
negativePromptstring否负面提示词
imagesarrayimg2img 时必填输入图片，每项为 {name, dataUrl} 对象

示例 1：文生图

curl -X POST http://your-host/openapi/v1/generate \ -H "Authorization: Bearer ak_xxxx" \ -H "Content-Type: application/json" \ -d '{ "prompt": "一只橘猫坐在窗台上，阳光洒进来", "mode": "txt2img", "modelId": "gpt-image", "n": 1, "hd": true, "aspectRatio": "16:9" }'

示例 2：图生图

curl -X POST http://your-host/openapi/v1/generate \ -H "Authorization: Bearer ak_xxxx" \ -H "Content-Type: application/json" \ -d '{ "prompt": "转为吉卜力动画风格", "mode": "img2img", "modelId": "gpt-image", "n": 1, "images": [ { "name": "photo.jpg", "dataUrl": "data:image/jpeg;base64,/9j/4AAQ..." } ] }'

images 中的 dataUrl 为完整的 Data URL（含 data:image/xxx;base64, 前缀），最多传 3 张图片。

响应格式

{ "success": true, "images": ["https://...output1.png"], "quota": { "limit": 200, "used": 15, "remaining": 185, "month": "2026-04" } }
