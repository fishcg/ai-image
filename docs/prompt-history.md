# 提示词历史功能使用说明

## 功能概述

提示词历史功能会自动保存你使用过的提示词（包括正面和负面 prompt），方便快速复用。

## 主要特性

- **自动保存**：每次成功生成图片后，系统会自动保存使用的提示词
- **智能去重**：相同的提示词会合并，并记录使用次数
- **快速复用**：点击历史记录即可一键填充到输入框
- **容量限制**：每个用户最多保存 100 条历史记录，超出后自动删除最旧的记录

## 使用方法

### 1. 查看历史

在生成页面（修图或生图模式），点击 Prompt 输入框上方的 **📝 历史** 按钮，会弹出历史记录下拉框。

### 2. 应用历史提示词

点击任意历史记录，系统会自动将该提示词（包括负面 prompt）填充到输入框中。

### 3. 删除历史记录

在历史记录右侧点击 **删除** 按钮，可以删除不需要的记录。

## 历史记录显示信息

每条历史记录显示：
- 提示词内容（最多显示 2 行）
- 负面提示词（如果有）
- 最后使用时间
- 使用次数（如果使用过多次）

## 技术实现

- 后端 API：`/api/prompt-history`
- 数据库表：`prompt_history`
- 自动保存时机：图片生成成功后
- 存储策略：相同提示词合并，更新使用次数和最后使用时间

## 注意事项

- 历史记录按最后使用时间倒序排列（最近使用的在最前面）
- 只有成功生成图片后才会保存提示词
- 历史记录是用户私有的，不会被其他用户看到
- 删除操作不可恢复，请谨慎操作
